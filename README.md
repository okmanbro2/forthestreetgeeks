# ForTheStreetGeeks
side project for making street maps + having cars like actually drive on them and stuff, pretty neat.
worked on in my free time; so lots of things aren't final and will be subject to change.

# FEATURES
- 3 sets of road types with 3+ sub-types
- ability to name your roads
- infinite space to make a city however big you want
